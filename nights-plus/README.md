# Nights Plus – Multi-Tenant Event Management SaaS

A multi-tenant SaaS platform enabling organizations to schedule and manage their own branded events.

<!-- <img width="1980" height="1458" alt="scorehub-mockup" src="https://github.com/user-attachments/assets/1d36cd6b-6540-4978-8aa8-4df855f85524" /> -->

## Features

- Serving as technical lead, defining the overall architecture and data model for a multi-tenant system.

- Designed database schemas supporting organizations, events, users, and permissions.

- Set up frontend and backend projects from the ground up with scalable patterns.

- Implemented a microservices-based backend architecture with async communication.

- Established testing standards including unit tests and end-to-end test coverage.

- Leveraged AI tooling and MCP servers (Bitbucket, Jira, Notion) via Claude Code to improve development speed and system understanding.

- Enhanced Claude Code capabilities by teaching it project-specific skills: for frontend, it knows how to write unit tests automatically on request; for backend, it can add new microservices, generate APIs, and write corresponding tests—enabling faster, AI-assisted development across the platform.

- Frontend: Next.js, TypeScript, React Hook Form, Zustand, Shadcn, Tailwind CSS, Payload CMS

- Backend: NestJS, Prisma, PostgreSQL, GraphQL, RabbitMQ

- Testing: Jest (unit), Playwright (E2E)
