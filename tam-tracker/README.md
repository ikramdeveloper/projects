## Features

- Worked on targeted feature development within an existing large-scale production system.

- Implemented subscription plan updates and changes, supporting multiple pricing tiers.

- Built an industry dashboard categorizing companies across 25 industries to analyze signal distribution.

- Developed industry-based alerting (daily & weekly) driven by signal volume and momentum.

- Assisted with database schema design and safe migrations for live users.

- Delivered features rapidly using AI-assisted development with Claude in a real-user production environment.

- Frontend: Next.js, TypeScript, Shadcn, Tailwind CSS

- Backend: Node.js, Hono, PostgreSQL, Drizzle ORM

- Infrastructure: AWS Lambda, Event Scheduler, SST
