## Features

- Built the complete backend using Node.js and Express with AWS-managed infrastructure.

- Designed an AI-led conversational flow where users evaluate business ideas across structured stages (vision, overview, execution, etc.).

- Implemented brainstorming and final evaluation phases generating actionable insights and reports.

- Integrated AI workflows using the OpenAI API with tool calls for guided questioning and structured responses.

- Modeled scalable data storage using DynamoDB.

- Focused exclusively on backend architecture and AI logic for a mobile-first product.

- Tech stack: Node.js, Express.js, DynamoDB

- AI: OpenAI Chat Completions & Responses APIs

- Infrastructure: AWS (EBS, managed services)
